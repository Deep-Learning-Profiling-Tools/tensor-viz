# TypeScript API Reference

The generated TypeDoc site covers `@tensor-viz/viewer-core`, which includes:

- the shared TypeScript session builders and manifest types
- the `TensorViewer` imperative browser API

The higher-level `viewer-demo` mount API is documented in the guide pages
because it is currently a very small iframe-backed surface rather than a large
reference API.

<a href="../_typedoc/index.html">Open the embedded TypeDoc reference</a>

Build it locally with:

```bash
npm run docs:ts
```
